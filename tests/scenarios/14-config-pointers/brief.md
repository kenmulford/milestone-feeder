# Brief: Danger button variant

Add a "danger" variant to the app's button component, for destructive actions like
delete and remove. It should look like the existing primary button but use the app's
danger color instead. Wire the new variant into the existing button component.

In scope:
- A `danger` variant on the existing button component, styled with the app's danger
  color from the design system.

Out of scope:
- Any new screen or page — this only extends the existing button component.
- Changing the primary or secondary variants.
